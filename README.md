# Impact-of-Leiden-Parameter-Choices
This is the code for Impact of Parameter Choice in Leiden Clustering of Single Cell Data, my final project for BINF 3360 at Baylor.
This repository contains code and data cloned from Leviyang's https://github.com/SLeviyang/resolutionTradeoffs in addition to the following
files, contained in the python folder:
- clustering.ipynb, which was used to explore the functions of Leviyang's code
- modularity_illustration.ipynb, which was used to generate the figure illustrating the concept of community modularity
-  ARI_parameter_sweep.ipynb, which contains code for the calculations and visualizations for the ARI analyses
-  cluster_res_parameter_sweep.ipynb, which contains code for the calculations and visualizations for the splitting resolution analyses
